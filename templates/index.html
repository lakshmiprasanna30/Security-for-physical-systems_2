About Us

We are committed to providing top-tier security solutions to protect your data, your components, and your digital ecosystem. Our Secure Component Manager is built to be intuitive, reliable, and highly effective in preventing cyber threats.



This project focuses on building a Python-based desktop security application that enhances system protection by monitoring and preventing multiple types of threats in real-time. It addresses critical concerns such as USB-based malware, phishing URL detection, unauthorized webcam access, and keystroke injection attacks. The application uses a modular structure where each component works independently yet interacts through a central GUI, built using Python‚Äôs Tkinter library. The user interface provides easy navigation for both technical and non-technical users, offering access to features like login authentication, hardware control, and threat notifications. USB devices are scanned immediately upon connection using MD5 hashing and VirusTotal API integration to identify malicious files. Clipboard monitoring is continuously active, and any copied URL is instantly checked by a trained Random Forest machine learning model to determine whether it‚Äôs legitimate or a phishing link. Webcam usage is monitored using system process tracking, and if unauthorized access is detected, alerts are shown and access is blocked. Keystroke behavior is also analyzed to detect rapid or suspicious typing that could indicate a hardware-based attack like USB Rubber Ducky. The system responds by disabling access to devices, locking the screen, or prompting for user authentication. Logging functionality is built in to track all system responses and user actions for future auditing. SQLite is used to manage user credentials securely. The combination of automated detection, user alerts, and device control ensures that the system remains protected at all times. It is suitable for use in homes, educational institutions, and small business environments. The project demonstrates how open-source tools and machine learning can be combined to build a lightweight yet powerful endpoint security¬†system

üîé Overview

In the rapidly evolving digital landscape, ensuring the security of computer systems has become a top priority. Malicious attacks targeting USB drives, phishing websites, webcam hijacking, and keystroke injection are increasingly common. This project delivers a comprehensive desktop application using Python that detects, prevents, and responds to these threats in real time.

The software is built with powerful system libraries and machine learning algorithms. It scans USB devices for potential malware, evaluates copied URLs for phishing threats, tracks unauthorized access to webcams, and detects fast, unnatural keystrokes that may indicate hacking attempts. Through an intuitive Tkinter-based GUI, users can perform security checks and respond to threats efficiently.

üß† Core Functionalities

  • Phishing URL Detection: Clipboard URLs are captured and checked using a trained machine learning model to prevent phishing attacks.
  • USB Malware Detection: USB insertion is detected instantly; files are scanned using hashing and verified via VirusTotal API.
  • Webcam Monitoring: Monitors real-time webcam usage and blocks unknown processes from accessing it.
  • Keystroke Injection Detection: Analyzes keystroke speed and patterns to detect scripted keystroke injections or hardware-based attacks.

üßÆ Algorithms Used

Random Forest: A robust supervised learning algorithm used to classify URLs into phishing or legitimate.

MD5 Hashing: Used to compare file hashes against known malicious files.

Heuristic-Based Keystroke Analysis: Detects unnatural typing speed to prevent automated attacks.
